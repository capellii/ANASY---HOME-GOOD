## 2. ESTRATÉGIA DE GO-TO-MARKET

### 2.1 EARLY ADOPTERS (Meses 1-6)

#### TARGET: Pessoas tech-savvy que já usam smart home

**Canais de Aquisição:**
1. Communities (10% do budget)
   - r/smarthome (Reddit)
   - Grupos Facebook "Casa Inteligente Brasil"
   - Telegram de automação
   - Fóruns de tecnologia

2. Influenciadores (20% do budget)
   - YouTubers de Tech/Automação (alcance 50k-500k)
   - Parcerias de produto (device em troca de review)
   - Micro-influenciadores em tech (ROI melhor)

3. Partnerships com Lojas (30% do budget)
   - AliExpress/Amazon (integração)
   - Lojas de eletrônicos (pontos de venda)
   - Instaladores de automação (comissão)

4. Organic/Blog (40% do budget)
   - SEO para "casa inteligente", "automação", "Matter"
   - Conteúdo: "Como automatizar sua casa em 2026"
   - Guias de integração (gratuitamente, com link)

**CAC (Customer Acquisition Cost) Esperado:** R$ 150-200
**LTV (Lifetime Value):** R$ 2.400 (24 meses × R$ 99,90 no Premium)
**Ratio LTV/CAC:** 12:1 ✓ (Muito saudável)

## 3. ESTRATÉGIA DE MVP (Minimum Viable Product)

### 3.1 MVP 0: CONCEPT VALIDATION (Semana 1-4)
**Objetivo:** Validar se o conceito tem tração

#### Deliverables:
- Landing page com vídeo 2min (pitch do ANASY)
- Email capture para waitlist
- Form de interesse (personas: executivo, casal, idoso)
- 1 vídeo demo de 5min (mockup de UI)

**Métricas de Sucesso:**
- 1.000+ emails na waitlist em 30 dias
- 200+ pessoas respondendo form
- 15% de taxa de conversão landing → waitlist

**Investimento:** R$ 5k (landing page + ads)
**Tempo:** 3-4 semanas

---

### 3.2 MVP 1: ALPHA FECHADO (Mês 1-3)
**Objetivo:** Validar tecnicamente com 50 usuários beta

#### Funcionalidades Mínimas:
```
BACKEND:
├─ Autenticação básica (email + password)
├─ API REST simples (CRUD de dispositivos)
├─ Suporte para 3 protocolos apenas:
│  • Zigbee (via zigbee2mqtt)
│  • Wi-Fi (HTTP POST simples)
│  • MQTT (básico)
└─ Database: PostgreSQL

HUB LOCAL:
├─ Raspberry Pi 4 como servidor (R$ 200)
├─ Docker para fácil setup
├─ Descoberta automática de dispositivos
└─ Sincronização com API (cloud)

APP:
├─ React Web (não mobile ainda)
├─ Dashboard com lista de dispositivos
├─ On/off para cada dispositivo
├─ Histórico de últimas 10 ações
└─ Suporte em português apenas

SEGURANÇA:
├─ Autenticação JWT
├─ Criptografia SSL/TLS
└─ Validação de entrada
```

**Recrutamento de Beta:**
- 30 usuarios: Comunidades online (gratuito)
- 20 usuarios: Colaboradores + família (recompensa R$ 100)

**Feedback Esperado:**
- Usabilidade: Onde clicam errado?
- Features: O que mais usam?
- Bugs: O que quebra?

**Investimento:** R$ 100k (3 devs × 2 meses)
**Resultado:** Versão estável, roadmap ajustado
